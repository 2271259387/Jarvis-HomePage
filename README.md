# homepage
用vue实现的JARVIS浏览器主页。    
使用three.js完成对jarvis主体和文字的渲染。  
在vue中使用three.js文字渲染模块的字体文件时需使用ES6的import导入。  
网站的jarvis还不是我理想中jarvis的样子，因为three.js局限性太高暂时做成这样。  
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
